# Unified Commerce 2nd Brain

This zip contains the **full folder structure** and **all markdown files already created**.

## Core platform assumptions

- Unified multi-tenant E-POS + E-Commerce
- Separate platform admin and tenant staff
- Tenant-specific customers and memberships
- Variant-level selling
- Inventory ledger + projection
- Returns and exchanges as separate documents
- Touchscreen-first, low-typing POS
- Feature access can be assigned by platform admin and then configured inside the tenant

## Usage rule

You do not need to fully write every file now.
Each markdown file is a prepared slot so you can expand only the files you need.
