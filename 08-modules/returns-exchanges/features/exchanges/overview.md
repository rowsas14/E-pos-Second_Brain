# Exchanges

## Purpose
Exchange documents built from a return, with new items, value difference, and collect/refund direction.

## Why this feature exists
Exchange is not just a return plus sale UI shortcut; it has its own value reconciliation and audit trail.

## Scope
- create exchange from approved return
- select new items
- collect difference or issue refund
