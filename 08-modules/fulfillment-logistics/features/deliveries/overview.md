# Deliveries

## Purpose
Shipment or pickup records attached to an order, allowing one order to split across multiple deliveries.

## Why this feature exists
Delivery execution should be separate from order header so partial shipment remains traceable.

## Scope
- create delivery
- assign order items/quantities
- mark packed, shipped, delivered, or returned
