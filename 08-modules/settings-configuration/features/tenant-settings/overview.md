# Tenant Settings

## Purpose
Generic scoped configuration store for tenant, outlet, or channel-specific behavior.

## Why this feature exists
The platform needs configurable behavior without constant code changes for each tenant.

## Scope
- create or update setting
- resolve setting at runtime by scope precedence
