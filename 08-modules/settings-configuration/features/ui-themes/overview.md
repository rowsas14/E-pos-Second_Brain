# UI Themes

## Purpose
Tenant-specific theme definitions for branding tokens such as colors, spacing, and typography.

## Why this feature exists
The BRD expects tenant-specific branding without developer-side CSS edits for each customer.

## Scope
- create theme
- set default theme
- preview and apply branding
