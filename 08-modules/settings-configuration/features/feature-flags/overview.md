# Feature Flags

## Purpose
Tenant-owned feature toggles with optional outlet or user scope plus config payload.

## Why this feature exists
Gradual rollout and tenant-specific capability control are required for a multi-tenant platform.

## Scope
- enable feature for tenant
- enable pilot feature for one outlet or user
- read resolved flag during runtime
