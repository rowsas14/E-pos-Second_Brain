# Payment & Return Reporting

## Purpose
Reporting views for payment breakdowns, refund totals, return volumes, and discount visibility.

## Why this feature exists
Finance and store operations need to see money movement and reverse-flow impact clearly.

## Scope
- review payment method mix
- review refund volume
- review return and discount exposure
