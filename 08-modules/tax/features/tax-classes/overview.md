# Tax Classes

## Purpose
Tenant-owned tax class master linked to products to describe tax behavior by goods/services/shipping context.

## Why this feature exists
Product-level tax assignment needs a business-friendly abstraction separate from raw tax rates.

## Scope
- create tax class
- assign tax class to products
