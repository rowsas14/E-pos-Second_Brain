# Staff Users

## Purpose
Tenant-bound staff identities used by tenant admins, managers, cashiers, inventory staff, and e-commerce operators.

## Why this feature exists
The revised operating model requires every staff user to belong to exactly one tenant. This prevents cross-tenant operator leakage.

## Scope
- create staff user
- activate, suspend, or deactivate user
- reset staff credentials and restore access
