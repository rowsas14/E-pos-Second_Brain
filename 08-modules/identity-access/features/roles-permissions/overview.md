# Roles & Permissions

## Purpose
Tenant and outlet scoped role definitions plus permission mapping used to authorize sensitive actions.

## Why this feature exists
POS discounts, refunds, stock adjustments, and administration all need explicit permission control rather than informal UI hiding.

## Scope
- create tenant role
- attach permission set to role
- assign tenant-scoped role to user
