# Discount Requests

## Purpose
Approval workflow for cashier or operator requested discounts on sale or order transactions.

## Why this feature exists
The business wants approval-based discounting rather than uncontrolled manual reductions.

## Scope
- raise discount request
- manager reviews request
- approved request flows into discount application
