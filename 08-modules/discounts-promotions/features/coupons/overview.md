# Coupons

## Purpose
Coupon master and redemption records with optional rule payload and usage caps.

## Why this feature exists
E-Commerce and promotional campaigns require reusable code-based discounting under tenant control.

## Scope
- create coupon
- activate coupon
- redeem coupon against order or sale where allowed
