# Carts

## Purpose
Active shopping carts for logged-in customers or guests with pricing snapshot inputs before order placement.

## Why this feature exists
E-Commerce requires an intermediate cart state that is separate from completed orders and supports guest checkout.

## Scope
- create cart
- add/update/remove items
- convert cart into order
