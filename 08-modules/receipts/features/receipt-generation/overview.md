# Receipt Generation

## Purpose
Creation of stored receipt/invoice output records for sale, order, return, and exchange documents.

## Why this feature exists
Operational documents need frozen render payloads for reprint, PDF, email, and audit reference.

## Scope
- generate receipt on document completion
- reprint or retrieve stored receipt
