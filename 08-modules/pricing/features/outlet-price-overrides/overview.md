# Outlet Price Overrides

## Purpose
Variant prices that override the normal list at a specific outlet when business policy requires location-specific pricing.

## Why this feature exists
Stores may need outlet-specific price differences while still using the same tenant catalog and price list infrastructure.

## Scope
- set outlet-specific price for a variant
- review override impact before activation
