# Tenant Onboarding

## Purpose
Creation of a new tenant business with its base identity, mode, currency, locale, default settings, and initial operating configuration.

## Why this feature exists
The platform must onboard new retail businesses quickly without mixing platform-side setup logic with tenant staff operations.

## Scope
- create tenant shell
- set base configuration and default theme/settings
- create first outlet and admin assignment handoff
