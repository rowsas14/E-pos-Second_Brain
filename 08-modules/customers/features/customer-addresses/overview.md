# Customer Addresses

## Purpose
Reusable billing and shipping addresses stored under a tenant-scoped customer.

## Why this feature exists
E-Commerce checkout and order snapshots need durable source addresses while letting customers maintain reusable address book entries.

## Scope
- add address
- edit address
- choose address during checkout and snapshot into order
